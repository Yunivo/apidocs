## Giriş {{ anchor: true }}

Bu dokümanda İleti Yönetim Sistemi İş Ortağı olan Dijiizin tarafından sunulan API'nin açıklamaları, güvenlik kısıtları ve kullanım örnekleri sunulmaktadır.
API, Farklı yazılım uygulamalarının birbirleriyle etkileşime geçmelerine olanak sağlayan servis çözümüdür.
Dijiizin API gelişen ve değişen ihtiyaçlara göre güncellenir ve güncel sürümleri müşterilerin  erişimine dokümantasyonlar üzerinden sunulur. Yapılan güncellemeler, https://apidocs.dijiizin.com üzerinden duyurulacaktır. Sunduğumuz API’yi kullanarak, Dijiizin üzerinden hizmet alan hizmet sağlayıcılar izinli verilerini İleti Yönetim Sistemine, yasal mevzuatlara uygun bir şekilde aktarabilecek, Aydınlatma yükümlülüğünü yerine getirdiğini kanıtlayabilecek ve Elektronik Açık Rıza toplayarak kanıt dosyaları ve logları edinebilecektir.
Tüm metotların HTTP istek ve cevap mesajlarında JSON söz dizim standardı kullanılmıştır. Güvenli API‘lerin erişiminde oAuth 2.0 protokolü takip edilmektedir. Bu doğrultuda oluşturulan access_token’lar Bearer token olarak kullanılarak İzin Yönetim, KVKK ve Açık Rıza istemlerinde güvenli veri alışverişi sağlanmaktadır. Kullanıcı bilgileriniz ile https://dijiizin.com/giris üzerinden sisteme giriş yaptıktan sonra, Hesap bölümüne girerek API Bilgileri bölümünden API Kullanıcısı oluşturmanız gerekmektedir.{{ className: 'lead' }}

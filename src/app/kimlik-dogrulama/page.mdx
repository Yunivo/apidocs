# Kimlik Doğrulama

Dijiizin.com üzerinde Ana kullanıcı ile https://dijiizin.com/user/api-bilgileri sayfasından API kullanıcısı oluşturulur. API şifresi, kullanıcı oluşturma esnasında bir defaya mahsus olarak oluşturulur ve size gösterilir. Eğer API şifrenizi kaydetmediyseniz, mevcut şifreyi yenileyebilir veya kullanıcınızı silerek yeniden oluşturabilirsiniz. API şifresi tarafınızdan belirlenemez. Şifrenize Dijiizin ekibi dahil, sizden başkası erişemez ve görüntüleyemez.

### Parametreler

Dijiizin panelinden oluşturduğunuz api kullanıcı adınız ve şifreyle access_token alınabilecektir. Çağrının sonucunda dönen mesajın bir örneği aşağıdaki gibidir.


```json
{
    "access_token": "eyJ0eXAiOiJKV1QiLCJhbGciO",
    "token_type": "bearer",
    "expires_in": 7200
}
```
